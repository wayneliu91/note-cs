# 背包和路径

## 题型说明

```wiki
重中之重，⼤⼚笔试、⾯试必考项、学习难但⾯试不难、掌握模型举⼀反三

DP专题：
• 专题：适⽤问题
• 专题：解题步骤
• 专题：最值、可⾏、计数三种类型	⼀些特殊⼩类别：树形DP、区间DP、数位DP
• 专题：空间优化

经典模型：
• 背包问题（0-1、完全、多重、⼆维费⽤、分组、有依赖的） • 路径问题
• 打家劫舍&股票买卖
• 爬楼梯问题
• 匹配问题（LCS、编辑距离）
• 其他（LIS）
```

## 动态规划解题过程

```wiki
1. 可⽤回溯解决：需要穷举搜索才能得到结果的问题（最值、可⾏、计数等）
2. 构建多阶段决策模型。看是否能将问题求解的过程分为多个阶段。
3. 查看是否存在重复⼦问题：是否有多个路径到达同⼀个状态。
4. 定义状态：也就是如何记录每⼀阶段的不重复状态。
5. 定义状态转移⽅程：也就是找到如何通过上⼀阶段的状态推导下⼀下阶段的状态。
6. 画状态转移表：辅助理解，验证正确性，确定状态转移的初始值。
7. 编写动态规划代码。

4和5两个步骤是难点，掌握的技巧就是：记忆【经典模型的状态】和【状态转移⽅程的定义⽅法】，举⼀反三。
```
